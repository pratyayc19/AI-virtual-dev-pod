🤖 AI Virtual Development Pod

A Multi-Agent AI System that simulates a complete software development team.
The system automatically converts business requirements → architecture → code → tests → review using coordinated AI agents.

Built using CrewAI, Groq LLaMA models, ChromaDB RAG, AWS S3, and Streamlit UI.

🚀 Features

✅ Multi-Agent Software Development Pipeline
✅ Automated Requirement → Code Generation
✅ Self-Critic Feedback Loop for code improvement
✅ RAG Memory System using ChromaDB
✅ Responsible AI Guardrails (safety + PII protection)
✅ Cloud Artifact Storage using AWS S3
✅ Interactive Streamlit Dashboard

| Agent               | Role                                                |
| ------------------- | --------------------------------------------------- |
| 🧑‍💼 Project Lead  | Orchestrates the pipeline and analyzes requirements |
| 📋 Business Analyst | Converts requirements into structured user stories  |
| 🎨 Design Agent     | Creates architecture, database schema, and APIs     |
| 💻 Developer Agent  | Generates source code from system design            |
| 🧪 Testing Agent    | Generates and evaluates test cases                  |
| 🔁 Critic Agent     | Reviews code quality and suggests improvements      |

🧠 Memory System (RAG)

The system uses ChromaDB as a vector memory store.

It stores artifacts such as:
user stories
system designs
generated code
test reports

When a new requirement is submitted, the system retrieves similar past artifacts to assist the agents.

Benefits:
knowledge reuse
contextual reasoning
improved generation consistency

🛡 Responsible AI Guardrails

To ensure safe usage, the system includes a Responsible AI layer.

Features:
blocks harmful prompts (e.g., hacking or malware generation)
detects sensitive information
redacts emails, phone numbers, and credit card numbers
validates both input and output

☁️ Cloud Infrastructure

The system uses cloud services for scalability.

LLM Access

Storage

AWS S3 stores artifacts generated by agents:
user stories
